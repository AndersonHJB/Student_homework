# 初识数据类型

| 关键词 | 特点                                                         |
| ------ | ------------------------------------------------------------ |
| type   | 检测数据类型                                                 |
| str    | 有序、从左到右：0 开始数<br>从右到左：-1 开始数、不可变      |
| list   | 有序、任意数据类型、[1, 2, 3, 4, 5, 6, 76, 78]、从左到右：0 开始、**可变** >>> word |
| tuple  | 有序、任意数据类型、**不可变**、>>> PDF、(1, 2, 3, 4, 5)     |
|        |                                                              |
